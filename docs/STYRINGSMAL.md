# Styringsmal

Bruk denne malen for prosjektstyring, migreringsnotater, hardware-profiler, og beslutningslogg.

## Formaal

- Hva produktet er
- Hva denne fasen skal levere
- Hva som eksplisitt ikke er i scope

## Naavaerende hardware

- Aktiv hardwareprofil
- Faktiske komponenter i bruk
- Kommunikasjonsvei
- Firmwareforutsetninger

## Naavaerende software

- Aktive runtime-flater
- Legacy/referanse-flater
- Delte bibliotek og kontrakter

## Maalarkitektur

- Maalprofil for hardware
- Planlagt klientflate
- Planlagt transportlag
- Delte domenelag

## Beslutninger

- Beslutning
- Status
- Dato
- Begrunnelse

## Aapne spoersmaal

- Uavklarte tekniske eller produktmessige forhold
- Hvem/eierskap hvis kjent
- Hva som blokkeres

## Risikoer

- Risiko
- Konsekvens
- Sannsynlighet
- Tiltak

## Test- og akseptkriterier

- Krav som ma oppfylles for a godkjenne fasen
- Manuelle tester
- Automatiske tester
- Hardware-verifisering

## Milepaeler

- Neste leveranser i rekkefolge
- Avhengigheter mellom dem

## Endringslogg

- Dato
- Endring
- Hvorfor
